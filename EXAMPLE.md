# Flujo esperado

- antes del login va a haber una pantalla de bienvenida al portal para afiliados
- se deben completar los datos del afiliado y su grupo familiar
- despues aparecera un modal que diga se debe esperar la activación del usuario por parte de un administrador
- el usuario no puede continuar hasta que un administrador active su cuenta
- se genera un token al iniciar sesión que nos servira para hacer las peticiones al backend
- una vez activado el usuario podra iniciar sesión
- lo lleva a la pantalla de inicio del portal para afiliados
